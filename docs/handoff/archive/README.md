# Handoff Archive

This folder is for superseded handoff, onboarding, and workflow documents.

Keep current handoff documents in `docs/handoff/`. When a document is replaced by a clearer or newer version, move the older version here instead of leaving multiple active copies with similar names.

Use date-prefixed filenames so the timeline stays clear:

```text
YYYY-MM-DD-short-description.md
```

If a document is only a temporary export or share copy outside the repo, do not add it here unless it records a decision worth preserving.
