MUSTERDECK
Rally, Command, Settle.
An independent operations console for Star Citizen crews. Find ops, plan ships, settle loot, run tournaments.
Free. Independent fan project. The social tools that we need.
MUSTERDECK

Multiple connected tools.
One console.

Rally Point

Find or publish operations

The public board for Star Citizen ops. Hosts post the roles and ships they need; players filter by activity, time zone, and ship, then apply. Browse free as a guest; sign up to post or apply.

Fleet Command

Plan ship composition, staff crews, run ops

The shared ops console for serious orgs. Build ship rosters and crew profiles, staff ops from saved templates, lock the roster when it starts, and follow a live ops view with a full audit log.

Coming soon
Proving Ground

Run tournaments and competitive events

Competitive events from signup to trophy. Single, double, round robin, or Swiss, from 1v1 to 5v5. Open signups, seed the field, publish brackets, and track standings and leaderboards live.

S.P.O.I.L.S.

Settle loot, payouts, ledgers

Settlement, Payouts, Operations, Inventory, Loot, and Shares. Log every haul, set the split, and it works out each member's share. Records only, run by your quartermaster.
