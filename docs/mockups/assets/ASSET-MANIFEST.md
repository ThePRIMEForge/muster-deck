# MusterDeck Mockup Asset Manifest

Use this manifest for mockup-only visual assets. When an asset is approved for production, copy its source notes into the production asset folder and keep this entry as history.

## Status Values

- Proposed: planned but not created.
- Draft: created for review.
- Approved: accepted for continued mockup use.
- Rejected: keep archived for history, but do not use in current mockups.
- Production candidate: approved for possible app implementation.
