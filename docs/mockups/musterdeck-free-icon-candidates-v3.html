MusterDeck Free Icon Candidates V3.5

Free and open-source candidates for the problem icons. This sheet uses existing icon libraries and local references so we can choose source families before any styling pass.

free-source proof v0.3.5
New ruleSource first, style second. Reuse our earlier custom symbols only where they already read well.
Preferred licensesMIT, ISC, Apache 2.0, CC0. These are low-friction for a fan web app.
Use carefullyCC BY is acceptable only if we track attribution in the app credits/license file.
Avoid for nowCC BY-SA, noncommercial, unclear marketplace terms, and logo/brand icon sets.
Production direction: use these clean SVGs as symbol sources, then apply MusterDeck personality through plates, distressed surfaces, mustard/olive/rust states, stencil labels, and panel textures.
